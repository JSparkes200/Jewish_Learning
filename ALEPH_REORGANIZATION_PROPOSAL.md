# Aleph Reorganization Proposal
## Thematic Groups · 5–10 Words Each · 6 Courses · 4 Tiers

---

## Design Principles

1. **Introduce → Game → Next** — One word at a time, with mini-games after each
2. **Group Review × 3** — After completing all words in a group, review them randomly 3 times before moving on
3. **5–10 words per group** — Small, digestible chunks (target ~7)
4. **+2–3 advanced words per group** — Each group has 2–3 harder/stretch words; these accumulate for the final section
5. **Aleph-4 is mandatory** — Culmination section: all accumulated advanced words + remaining vocabulary
6. **Adaptive practice sentences** — Games randomly populate with words the user has already learned
7. **Find the Correct Sentence** — New game: 4 sentences, 3 wrong (word/grammar errors), 1 correct
8. **Tier 1→4** — Each tier increases difficulty (simpler → more abstract, more grammar)

2. **Mini-game** with word 1 (e.g. multiple choice)
3. **Introduce** word 2 → game

## Adaptive Practice Sentences

**Principle:** All sentence-based games (Fill-in, Translate, Find the Correct Sentence) **only use words the user has already learned** in completed groups.

### Implementation

- **Learned pool:** `completedGroups` → flat list of all words from those groups
- **Sentence templates:** Predefined structures with slots, e.g.  
  `אֲנִי ___ סֵפֶר` (slot = verb: read, love, write)
- **Slot fillers:** Randomly pick from learned pool that fits the slot (verb, noun, adj, etc.)
- **Progression:** Early groups → very simple slots (e.g. "I ___ bread" → אוכל). Later → richer, more varied sentences.
- **Optional:** Tag words in D with `pos` (verb/noun/adj/etc.) for smarter slot matching.

### Example templates (expandable)

| Template (Hebrew) | Slot type | Example fillers (once learned) |
|-------------------|-----------|--------------------------------|
| אֲנִי ___ כָּל יוֹם | verb | אוֹהֵב, קוֹרֵא, הוֹלֵךְ |
| זֶה ___ יָפֶה | noun | בַּיִת, סֵפֶר, כֶּלֶב |
| הַ___ גָּדוֹל | noun | כֶּלֶב, בַּיִת, יֶלֶד |
| יֵשׁ לִי ___ | noun | סֵפֶר, כֶּלֶב, חָבֵר |
| אֵין ___ | noun | מַיִם, לֶחֶם, כֶּסֶף |

## New Game: Find the Correct Sentence

**Concept:** User sees 4 Hebrew sentences. Three contain errors (wrong word, wrong grammar, or both). One is correct. User must tap the correct one.

### Error types (by difficulty)

| Stage | Error types | Examples |
|-------|-------------|----------|
| Early (Tier 1–2) | Wrong word, obvious gender mismatch | כֶּלֶב גְּדוֹלָה (should be גָּדוֹל) |
| Mid (Tier 2–3) | Wrong preposition, wrong plural | עַל הַשֻׁלְחָן vs בַּשֻׁלְחָן |
| Late (Tier 3–4) | Wrong verb form, subtle article | הָלַכְתִּי vs הָלַכְתָּ |

### Generation

- **Correct sentence:** From template bank, filled with learned words. Ensure grammar is correct.
- **Wrong sentences:** Same template, but inject one error:
  - Swap a word for a wrong one (same POS, different meaning)
  - Wrong gender/number agreement
  - Wrong preposition
  - Wrong definite article
- **Shuffle** the 4 so correct position is random.

### Feedback on wrong choice

- "Not quite. The error was: **gender agreement** — כֶּלֶב is masculine, so the adjective should be גָּדוֹל, not גְּדוֹלָה."
- Short, teachable explanation. Reinforces grammar.

### Suggested placement

- Unlock after Group 5.1 (Gender Agreement grammar)
- Appear in rotation with MC, Fill-in, Match
- 3–5 questions per session, or as "Grammar Check" mini-game between groups

## Additional Suggestions

### 1. Spaced group review

- Every 5 completed groups → trigger a "Mixed Review" session: random selection from all learned words.
- Prevents early groups from fading. Option to make this automatic or user-triggered.

### 2. Sentence-building game (stretch)

- Given English: "I love my dog"
- User assembles from a **word bank of learned words**: אֲנִי, אוֹהֵב, הַכֶּלֶב, שֶׁלִּי, etc.
- Drag/drop or tap in order. Validates word order and שֶׁל construction.
- More complex; good for late Aleph / early Bet.

### 3. Listen & identify

- TTS speaks a sentence. User sees 4 written options, picks the one that matches.
- Combines listening with reading. Uses learned words only.

### 4. Difficulty scaling in "Find the Correct Sentence"

- Tier 1–2: One obvious error per wrong option
- Tier 3–4: Two errors, or subtler mistakes (e.g. ה vs no article)

### 5. Predefined correct sentences per word

- Each vocabulary word has 1–2 "canonical" sentences where it appears correctly.
- Used for: (a) introduction phase, (b) generating correct option in Find the Correct Sentence, (c) TTS examples.
- Stored as `{ wordKey: [{ h: "…", e: "…" }] }` in data.

### 6. Grammar interlude pacing

- After groups 5.1, 6.2, 8.1, 10.2, 10.3: Show 3–4 grammar quiz items.
- Then "Find the Correct Sentence" can draw from those grammar concepts in subsequent sessions.
